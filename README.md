# Money Tracker App 💰

A simple web application to track your personal transactions — borrowings, loans, and returns — built with React and Flask.

---

## Features

- Add, update, delete transactions  
- View a list of all transactions with details  
- Sort transactions by date  
- Responsive and easy-to-use UI  
- Backend API with Flask and DynamoDB (or any other DB)  
- JWT-based authentication support (optional)  

---

## Tech Stack

- **Frontend:** React (Vite), Axios, Tailwind CSS  
- **Backend:** Flask, Python, Boto3 (for DynamoDB)  
- **Authentication:** JWT with cookies  
- **Deployment:** Localhost / AWS (optional)  

---
